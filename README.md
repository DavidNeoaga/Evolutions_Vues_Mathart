Animation de courbe avec Manim  

Ce projet contient le code source d'une animation mathématique réalisée avec la bibliothèque Python Manim.

C'est mon tout premier script ! J'ai créé cette animation pour visualiser une courbe de vues spécifique utilisée dans la vidéo YouTube de Mathart : (https://youtu.be/yyyN_UMvQik?t=72) à 1:12.

J'ai toujours voulu apporter ma "patte" et me démarquer des autres monteurs en utilisant des assets uniques créés sur mesure. Manim m'a permis de réaliser exactement ce que j'avais en tête, là où des templates classiques auraient été limités.

Même si j'ai passé une journée entière dessus pour le peaufiner, je suis très content du résultat final !

Technologies
- Python 3.10+
- Manim
- NumPy

Comment lancer le code
Si vous voulez tester le code chez vous :
1. Installez Manim (`pip install manim`)
2. Lancez la commande :
   ```bash
   manim -pql EvolutionVues.py EvolutionVues
